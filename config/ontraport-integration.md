# Ontraport Integration

Documentation for linking Shopify to Ontraport.

## Integration Steps

1. **Prerequisites**
   - Shopify store fully launched
   - Ontraport account access
   - API credentials configured

2. **Connection Setup**
   - Install Ontraport app in Shopify
   - Configure webhooks for order events
   - Map customer fields between systems

3. **Automation Workflows**
   - Welcome email sequence
   - Order confirmation automation
   - Abandoned cart recovery
   - Post-purchase follow-up
