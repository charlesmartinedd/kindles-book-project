# Kindle's Book Project

Alexandria's Design - Book Publishing Initiative

## Overview

This repository contains resources and materials for Kindle's book publishing project with Moosepack brand. The project includes Shopify store setup and integration with Ontraport.

## Project Details

**Brand:** Moosepack
**Project Owner:** Kindle
**Brand Assets Contact:** Jessie Eatzler

## Workflow

1. **Brand Identity Setup**
   - Jessie Eatzler will send brand identity materials
   - Awaiting brand assets and guidelines for Moosepack

2. **Shopify Launch**
   - Launch Shopify account for Moosepack
   - Configure store with brand identity
   - Set up products and payment processing

3. **Ontraport Integration**
   - Link Shopify to Ontraport after store launch
   - Configure customer automation and email marketing
   - Set up sales funnel integration
